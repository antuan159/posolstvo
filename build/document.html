Список необходимых документов для подачи на рабочую (национальную) литовскую Д визу:

  1. Загран паспорт ( если есть 2 загран, он тоже нужен брать обязательно)

  2. Украинский паспорт

  3. Фотография (1 шт.), размер 3.5мм * 4.5мм, 80% лица, цветная, на белом фоне, без уголка! Фото должно быть не более месячной давности

  4. Страховка ( можно сделать у нас возле посольства)

  5. Анкета ( можно сделать у нас возле посольства)

  6. Права (для водителей), диплом (для строителей и т.д.). Перевод на литовский язык.

  7. Электронный код приглашения!( ДАЁТ РАБОТОДАТЕЛЬ!) Работодатель получает его на сайте EPIS

  8. Трудовой договор на литовском языке

  9. Трудовой договор на русском языке (не обязательно)

Для остальных профессий, которые не попадают под облегчённую систему виз, нужен оригинал разрешения на работу (LIETUVOS DARBO BIRZA).
